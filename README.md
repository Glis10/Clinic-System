INTRODUCTION:

The Clinic Appointment System is a web-based application developed using PHP and MySQL.
The main objective of this system is to manage clinic operations such as doctor records, patient records, and appointment scheduling in an organized and efficient way.

This project is designed as a college-level web application project focusing on server-side scripting, database connectivity, authentication, and CRUD operations.


SETUP INSTRUCTIONS:

 -  Log in as admin to access the dashboard.
 -  The top menu lets you navigate to Patients, Doctors, Appointments, and Search.
 -  The welcome section confirms the logged-in user.
 -  Summary cards show:
 -  Total Patients
 -  Total Doctors
 -  Total Appointments
 -  Today’s Appointments
 -  Today’s Appointments shows appointments scheduled for today.
 -  Upcoming Appointments shows future bookings.
 -  Click View All to see full appointment lists.
 -  Use Logout to safely exit the system.


FEATURES:

 -  User authentication (login and logout)
 -  Doctor management
 -  Patient management
 -  Appointment scheduling
 -  Appointment slot handling
 -  Search functionality with filters
 -  Clean and simple user interface
 -  Live check for available time slots
 -  Prevention of overlapping schedules(1 Hour difference)

 KNOWN ISSUES:
 -  Password reset functionality is not available.
 -  User interface is basic and not fully optimized for all screen sizes.
